# Media Server Config

- Jellyfin + Jellyseer (media browsing)
- qBittorrent + Gluetun (VPN-secured downloads)
- Radarr / Sonarr / Lidarr / Readarr / Prowlarr / Jackett (media automation)
- Traefik + AdGuard + ntfy (routing, DNS, notifications)
